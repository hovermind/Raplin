## Variable
`var` keyword to declare variable
```
var x: Int = 10;

var y: Double;
y= 13.5
```
`val` keyword to declare read-only variable (also for constant)
```
val PI = 3.1416;
```

## Datatypes
Starts with Capital letter & every type has helper methods (similar to C#)

* Int (default), Long
* Float, Double (default)
* Byte, Short
* Boolean
* Char
* String
