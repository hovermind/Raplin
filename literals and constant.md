## Constants
No keyword for constant, `val` is used for both constant and read-only variables
```
val PI = 3.1416;
```

## Literals
**default:** `Int` & `Double` (no suffix needed)   

**Type suffix:**   
* `Long` => L 
* `Float` => F
```
var myLong = 100000000005L
var myFloat = 3.1416238458152F
```
